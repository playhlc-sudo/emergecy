⚙️ API URL 설정

Google Apps Script 웹 앱 URL을 입력하세요.

예: https://script.google.com/macros/s/AKfycbxxx.../exec
💡 API URL 받는 방법:
1. 제공받은 스프레드시트 "사본 만들기"
2. 확장 프로그램 → Apps Script 클릭
3. 배포 → 새 배포 → 웹 앱
4. 액세스 권한: 모든 사용자
5. 웹 앱 URL 복사
